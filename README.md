# 🎬 Movie Recommendation System

## 📌 Project Overview
This is a simple Machine Learning project that recommends movies based on similarity of genres.

## 🧠 How it works
- Uses TF-IDF Vectorization to convert movie genres into numerical form
- Uses Cosine Similarity to measure similarity between movies
- Recommends top 5 similar movies based on input

## 📂 Dataset
- movies.csv
- ratings.csv

## ⚙️ Technologies Used
- Python
- Pandas
- Scikit-learn
